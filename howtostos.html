Flashing And Booting

Prerequisites:

# Usb flash drive

# Chromebook

# Computer that is not school-sanctioned

# A Terra OS image

Flashing

I reccomend a usb that's at least 30gb in size.

If you plan on actually using this often, get at least 64gb, and to play games 120gb is reccomended.

First, boot up your computer (not chromebook) and go Here for windows pc's

If you can't use Rufus (on linux, mac or otherwize) I recommend Balena Etcher

Start by launching one of the provided flashing utilities

Then flash the usb drive with the Terra OS image, eject the usb drive, and Start Booting
